# Denim Gartex — Public Website

Marketing website for Denim Gartex: garment washing, denim finishing, and production capabilities.

The business admin app (clients, lots, invoices, payroll) lives in a separate repo: [denim-gartex-app](https://github.com/devohmm/denim-gartex-app).

## Scripts

- `dev` — start local dev server
- `build` — production build
- `preview` — preview production build
- `lint` — run ESLint

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Build and deploy the `dist/` folder to any static host (e.g., Netlify, Vercel, GitHub Pages).
