# Denim Gartex — Denim Processing Unit

A React (Vite) website for Denim Gartex with contact details and specialties.

## Scripts
- dev: start local dev server
- build: production build
- preview: preview production build
- lint: run ESLint

## Run locally
1. Install dependencies
2. Start the dev server

## Deploy
Build and deploy the `dist/` folder to any static host (e.g., Netlify, Vercel, GitHub Pages).
